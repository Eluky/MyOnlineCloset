# MyOnlineCloset
A tool which you can manage your closes online and mark your inspiration anytime

## 一、产品概述
- 1.1 产品定位
产品名称：我的电子衣橱
平台：微信小程序
一句话概括：
一个帮助用户线上记录用户拥有的服装及搭配，并且可以记录用户搭配灵感的小程序。
- 1.2 产品目标
管理：提供服装管理的功能，为用户提供一个线上管理助手。
记录：支持用户建立心愿单，记录搭配灵感和待购买服装心愿单。
## 二、用户与使用场景
- 2.1 目标用户
所有有时装搭配需求的个人或企业。
- 2.2 核心使用场景
形象设计：专业的形象设计公司或者造型师记录服装搭配案例。
个人服饰记录：个人记录自己的日常服装及搭配或搭配灵感。
## 三、整体信息架构 & 导航
- 3.1 底部导航（Tab）结构
底部固定五大 Tab：
单品
搭配
灵感
心愿单
我的
切换规则：点击 Tab 立即切换对应一级页面，保持页面状态（如选中的词单品、滚动位置）。
